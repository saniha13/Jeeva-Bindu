# Jeeva-Bindu – Rapid Response Blood Donor App

## Project Overview
Jeeva-Bindu is a healthcare-focused Android application developed using Kotlin and Jetpack Compose. The app helps hospitals quickly find compatible blood donors during emergencies through a location-based rapid response system.

## Problem Statement
In taluka-level hospitals, finding blood donors for rare blood groups during emergencies is difficult. Existing communication methods like WhatsApp are slow and unreliable.

## Features
- Donor registration system
- Blood group compatibility matching
- Emergency blood alerts
- "I'm Coming" response feature
- Hospital directory
- Google Maps integration
- Donor eligibility tracker
- Location-based filtering

## Technologies Used
- Kotlin
- Jetpack Compose
- Material 3
- Firebase Cloud Messaging (Concept)
- Google Maps Intent
- Android Studio

## App Workflow
1. User registers as donor
2. Emergency alert is generated
3. Compatible donors are filtered
4. Donor responds using “I’m Coming”
5. Hospital location opens in Google Maps

## Medical Compatibility Logic
The app uses real blood compatibility rules to show only valid donor matches.

## Success Criteria Achieved
- Emergency alerts simulated
- 90-day eligibility calculation implemented
- Emergency UI uses Red color
- Ready-to-donate UI uses Green color

## Installation Steps
1. Clone repository
2. Open in Android Studio
3. Sync Gradle
4. Run on emulator/device
